# Tailscale Engineering Lab: Windows & iOS Integration
This repository documents my personal homelab project designed to solve remote management challenges using Tailscale. My goal was to build a secure, high-performance "head-less" environment on Windows and iOS, prioritizing Direct Peer-to-Peer (P2P) connectivity and Zero-Trust security.
